How to Draw:

1. Start off by selecting a
shape button

2. Selected a colour from the
colour palette

3. Drag on the WHITE canvas
to draw your shape in the
colour you chose

4. If you want to undo a shape
click the 'Undo' button. If you
want to start from the
beginning with a clean
canvas click the 'Reset' button.

5. If you want to use the brush
tool then you can change
the width of the brush by
selecting a brush width below
the brush button.

6. Have fun and create your
masterpiece!
